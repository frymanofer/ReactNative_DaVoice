# React Native Speech to Text On Device

If you are searching for **React Native speech to text**, **React Native STT**, **React Native ASR**, or **offline speech to text for React Native**, this repository shows how speech recognition fits into a broader on-device voice experience.

The example app transitions into speech recognition after <a href="https://davoice.io/wake-word" target="_blank" rel="noopener noreferrer">wake-word detection</a> and can optionally enforce the onboarded <a href="https://davoice.io/speaker-recognition" target="_blank" rel="noopener noreferrer">speaker</a>. It also demonstrates a practical pause between stages so wake word and ASR do not fight over the audio pipeline.

This is especially useful for teams building:

- voice assistants
- hands-free app flows
- healthcare voice workflows
- field-service apps
- privacy-sensitive products
